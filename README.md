# Broadcast Relay Magic

Broadcasts traffic across tunnel, allowing LAN games to play over tunnels
(such as Tailscale).

## Dependencies

- Legacy Windows - [WinPcap](https://www.winpcap.org/)
- Modern Windows - [Npcap](https://npcap.com/)

## Invocation

```
python brm.py <config_name>
```
e.g.:
```
python brm.py a
```
